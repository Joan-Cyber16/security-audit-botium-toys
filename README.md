# Security Audit – Botium Toys 🔐

## 📌 Overview
This project is a simulated cybersecurity audit conducted for Botium Toys. The objective is to evaluate the organization's security posture, identify risks, and recommend improvements based on industry standards.

## 🎯 Objectives
- Identify and assess organizational assets
- Evaluate existing security controls
- Analyze compliance with regulations (PCI DSS, GDPR, SOC)
- Provide actionable security recommendations

## ⚠️ Risk Summary
- Risk Score: 8/10 (High)
- Main Issue: Lack of asset management and security controls
- Potential Impact: Data breaches, regulatory fines, operational disruption :contentReference[oaicite:0]{index=0}

## 🛡️ Key Findings
- No encryption for sensitive data
- No intrusion detection system (IDS)
- No backups or disaster recovery plan
- Weak password policies
- Excessive access to sensitive data

## 📂 Project Structure
- Risk Assessment
- Controls Assessment
- Compliance Checklist
- Security Recommendations

## 📚 Framework Used
- NIST Cybersecurity Framework (CSF)
